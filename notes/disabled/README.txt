#
# Notes files placed here are ignored by POE2 Overlay Timer.
#
# To switch between note sets:
#   - Move the file you want ACTIVE into the parent notes/ folder
#   - Move the file you want INACTIVE into this disabled/ folder
#
# Example:
#   Want English notes  → keep tip_with_050_EN.txt in notes/
#                          move tip_with_050.txt    here
#   Want Chinese notes  → keep tip_with_050.txt     in notes/
#                          move tip_with_050_EN.txt  here
