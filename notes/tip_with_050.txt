# ────────── Act 1 ──────────


[Clearfell / 皆伐]
{
1-1 沿邊找神秘營地拿1級技能石；沿中間路找到貝伊拉拿10%冰冷抗性
Next -> 葛瑞爾林
}

[The Grelwood / 葛瑞爾林]
{
1-2 大鍋爐女巫拿1級補助石；進繞陰林踩傳點，踩大樹傳點，赤谷入口在荊棘怪旁邊
Next -> 赤谷
}

[The Red Vale / 赤谷]
{
1-3 打3個柱子拿任務物品到葛瑞爾林解放黑衣幽魂，回城找烏娜，進纏繞陰林
Next -> 纏縛陰林
}

[The Grim Tangle / 纏縛陰林]
{
1-4 出口看大地圖中不朽帝國之墓位置在左下，圖中出口相對位置就在左下，反之亦然
Next -> 不朽帝國之墓
}

[Cemetery of the Eternals / 不朽帝國之墓]
{
1-5 找配偶和執政官墓室
Next -> 配偶的墓室
}

[Tomb of the Consort / 配偶的墓室]
{
1-6 找征戰的寶庫，打精英拿1級補助石
Next -> 政務官陵墓
}

[Mausoleum of the Praetor / 政務官陵墓]
{
1-7 BOSS房間一般在入口對角。拿完任務物品回不朽帝國之墓傳送點點開大門，進去打BOSS
Next -> 獵場
}

[Hunting Grounds / 獵場]
{
1-8 沿著邊走，打樹靈祭祀拿1級補助石；打鴉鈴拿+2天賦書；找到弗雷索恩傳送點
提示: 升到9級
Next -> 奧格姆農地
}

[Ogham Farmlands / 奧格姆農地]
{
1-9 沿著大路走找烏娜的小屋，拿烏娜的琴。沿著大路走到底，進入奧格姆村踩傳送點，回到營地更新血瓶藍藥、檢查裝備。傳送獵場，記錄點傳送弗雷索恩
提示: 升到10級
Next -> 弗雷索恩
}

[Freythorn / 弗雷索恩]
{
1-10 先打3個小祭壇，打大祭壇BOSS拿30精魂。
Next -> 奧格姆村
}

[Ogham Village / 奧格姆村]
{
1-11 出來入口後往斜上方走（自己悟），找到BOSS入口記錄點，如中途沒有遇到任務物品，返回入口傳送點，往下方走找到任務物品。打完奧格姆村BOSS，解救NPC
Next -> 宅第壁壘
}

[The Manor Ramparts / 宅第壁壘]
{
1-12 一般成反 C 字形，沿著 C 字走一圈，找到絞刑架拿1級補助石，繼續走，到奧格姆宅邸入口
提示: 12級半或13級
Next -> 奧格姆宅第
}

[Ogham Manor / 奧格姆宅第]
{
1-13 一層沿邊走，需要找到燭光 BOSS +20最大生命，傷害夠可以先打，傷害不夠14級再來打。一層二層為樓梯結構，二層入口在一層左下，那麼三層入口就在二層右上。BOSS房一般在三層左上。打完尾王回城，交完所有任務，找黑衣幽魂進入第二章 
提示: 王前14級
}

# ────────── Act 2 ──────────


[Vastiri Outskirts / 瓦斯提里郊區]
{
2-1 沿邊走，找到野外營地，寶箱出2級補助石。繼續沿邊走，找到BOSS。回城對話
Next -> 莫頓挖石場
}

[Mawdun Quarry / 莫頓挖石場]
{
2-2 延邊走，或者指著對角走，此處很容易爆護甲片，萌新可以稍作停留打怪
Next -> 莫頓礦坑
}

[Mawdun Mine / 莫頓礦坑]
{
2-3 BOSS房在左上角，沿著鐵路走。打完回城對話
Next -> 叛徒之路
}

[Traitor's Passage / 叛徒之路]
{
2-4 地形複雜，有很多種形狀，沿著最遠處走基本能走通，中途遇到石碑岔路口，貼有符文的路口就可以進入芭芭拉的BOSS房，打完領取巨靈幣。然後沿邊走，走不通就返回路口走另外一條路。全體地圖會經歷3到5次窄路上坡，找到窄路上坡就是正確的路
Next -> 哈拉妮關口
}

[The Halani Gates / 哈拉妮關口]
{
2-5 一條通路，注意在中途的記錄點重生一下，讓 NPC 過來。打完關口BOSS，進入死路回城 
提示: 最低升到 18 級
Next -> 乳齒象惡地
}

[Mastodon Badlands / 乳齒象惡地]
{
2-6 此時可以選擇進入絲克瑪試煉，沒有需求的就直接進入乳齒象惡地，延邊走，找到骸骨神殿，拿2級補助石。繼續延邊走，進入骨坑
Next -> 骨坑
}

[The Bone Pits / 骨坑]
{
2-7 地圖沿著出口直走到底，一般會遇到記錄點，憑運氣分左右走，找到 BOSS 房。過程中多打狗，掉落任務物品。打完回城
提示: 太陽部落聖物
Next -> 凱斯城
}

[Keth / 凱斯城]
{
2-8 沿邊走，找下出口，找卡巴拉一般在地圖中間偏兩邊位置，這邊沒找到就去另外一邊。打死拿到+2天賦書
Next -> 失落之城
}

[The Lost City / 失落之城]
{
2-9 先直走到底，看運氣分左右，然後繼續看運氣往下或者往上走，沿邊找到下一層入口。此時檢查自己是否掉落任務物品，沒掉落的話回到凱斯城繼續殺怪直到掉落任務物品 
提示: 太陽部落聖物
Next -> 掩埋神殿
}

[Buried Shrines / 掩埋神殿]
{
2-10 延邊走，找到元素奉獻神殿，選擇戒指（建議火）。繼續延邊走找到 BOSS 房入口。賽季玩法可能掉低階工匠石。打 BOSS，點燃女神，回城交任務
提示: 護符欄位
Next -> 泰坦之谷
}

[Valley of the Titans / 泰坦之谷]
{
2-11 先沿路走，對直走，找到下一層入口，然後延邊走，找到 3 個任務點，以及一個傳送點。骨坑和凱斯城的任務物品放到傳送點的祭台。進入泰坦之谷，泰坦之谷整體是長方形，先走長邊，然後憑運氣沿邊找到 BOSS。打完 BOSS 回城
提示: 護符
Next -> 戴斯哈
}

[Deshar / 戴斯哈]
{
2-12 先往前直走一段距離，進入住房區，沿邊走，找到下一層入口。然後繼續找任務物品遺書 (+2 天賦)，一般會出現在某個房子的下方
提示: 25 級
Next -> 悼念之路
}

[Path of Mourning / 悼念之路]
{
2-13 總體沿著一個方向走，進入下一關
Next -> 戴斯哈尖塔
}

[The Spires of Deshar / 戴斯哈尖塔]
{
2-14 總體沿著一個方向走，找到 BOSS 房，還得在中間某個位置找到記錄點，點任務物品拿+10%電抗 
提示: 26 級
Next -> 無畏隊
}

[The Dreadnought / 無畏隊]
{
2-15 新賽季無畏隊只有一個圖。打尾王，回城，跟著指引，進入第三章 
提示: 最好升到 29 級

# ────────── Act 3 ──────────


[Sandswept Marsh / 風沙沼澤]
{
3-1 沿邊走，找到根淤帶拿 9 級技能石（可跳過）。找到歐洛克營地開寶箱拿低階工匠石，賽季玩法出第一個 3 級補助石。繼續延邊走找到出口
Next -> 叢林遺跡
}

[Jungle Ruins / 叢林遺跡]
{
3-2 沿路走找到傳送點進入劇毒墓穴。
Next -> 劇毒墓穴
}

[The Venom Crypts / 劇毒墓穴]
{
3-3 沿路走找到傳送點進入劇毒墓穴。沿邊走找任務物品與 3 級補助石寶箱（兩者通常相對）。拿完任務物品回業林遺跡繼續沿地圖走，打死猿神拿 +2 天賦書。下一關入口在猿神四周，找到水道入口 
提示: 劇毒藥水
Next -> 感染荒地
}

[Infested Barrens / 感染荒地]
{
3-4 直走找到傳送點，然後沿邊走，找到龍溪濕地入口 
提示: 阿扎克泥沼30精魂
Next -> 龍蜥濕地
}

[Chimeral Wetlands / 龍蜥濕地]
{
3-5 沿邊走，找到混沌神殿和 BOSS 房。打完 BOSS 進入下一關 
提示: 32 級
Next -> 吉卡尼的機械迷城
}

[Jiquani's Machinarium / 吉卡尼的機械迷城]
{
3-6 先去右方找核心（70%機率在右），找不到再去左。用位移技能過第一段路，走中間尋找2個核心（注意：此時點記錄點重生核心會消失）。沿通道找到出口，王房通常在出口對面。打死王拿 +10% 火抗
Next -> 吉卡尼的聖域
}

[Jiquani's Sanctum / 吉卡尼的聖域]
{
3-7 先找左右兩邊記錄點，可在此重生刷新任務物品位置並裝在發電機上。然後重生回入口往前打王拿任務物品。傳送到業林遺跡，在水道入口放核心進入
Next -> 阿札克泥沼
}

[The Azak Bog / 阿札克泥沼]
{
3-8 進入泥沼，王一般在右上方，打死拿任務物品。
Next -> 瑪特蘭水道
}

[The Matlan Waterways / 瑪特蘭水道]
{
3-10 通關水道回到主城，往下對話艾瓦，進入淹沒之城
Next -> 淹沒之城
}

[The Drowned City / 淹沒之城]
{
3-11 到傳送點後往下沿石板路走，若左下路斷說明無寶庫。經記錄點直走分左右，一邊是熔岩寶庫，一邊是出口。先去熔岩寶庫
Next -> 熔岩寶庫
}

[The Molten Vault / 熔岩寶庫]
{
3-12 整體順時針走，BOSS 在右下，打完拿任務物品。回到淹沒之城前往下一關 
提示: 36 級
Next -> 污垢頂峰
}

[Apex of Filth / 污垢頂峰]
{
3-13 過第一個記錄點，岔路先往左上走（70%王在左上），若無則沿頂部找（王在右上）。打完拿任務物品。回淹沒之城往入口走進營地，直走在神廟入口放物品進入
Next -> 科佩克神殿
}

[Temple of Kopec / 科佩克神殿]
{
3-14 地圖成三角形，下層入口在三個角。賭輸可記錄點重生換點。打 BOSS，對話後等艾瓦出錯回城，進時空門往下走
Next -> 奧札爾
}

[Utzaal / 奧札爾]
{
3-15 沿石板路走找到 BOSS 幹掉。沒石板路就是錯路 
提示: 犧牲之心(阿戈拉也有)
Next -> 阿戈拉

[Aggorat / 阿戈拉]

提示: 39 級
Next -> 漆黑密室

[Isle of Kin / 金氏島]

}

[Kedge Bay / 凱吉灣]
{
4-3 往最遠走進入旅程終末。打王、回城找丹尼格拿任務物品，回傳送門做任務打怪。回城找下方圓貞拿 +2 天賦點
Next -> 伯勞鳥之島
}

[Shrike Island / 伯勞鳥之島]
{
4-4 直走打王，對話 NPC 回船，進入悉尼寇拉之眼
Next -> 悉妮蔻拉之眼
}

[Eye of Hinekora / 悉妮蔻拉之眼]
{
4-5 一條路，打完 3 個關口往上走，找到娜瓦莉憩地拿最大魔力上限獎勵
Next -> 亡者之殿

[Halls of the Dead / 亡者之殿]
{
4-6 左上、右上、右下拿 3 個刺青（一般選加抗性）。右下打完往右打 BOSS 拿任務物品，往上走
Next -> 祖靈的試煉

[Trial of the Ancestors / 祖靈的試煉]

[Whakapanu Island / 瓦卡帕努島]
{
4-8 沿內圈走找沙地，多半能找下關入口；找不到則沿外圈邊找
Next -> 吟謠洞窟
}

[Singing Caverns / 吟謠洞窟]
{
4-9 王在右上、左上或中間。珍珠項鏈在王房對角。打完回船
Next -> 廢棄監獄
}

[Abandoned Prison / 廢棄監獄]
{
4-10 入口在右下/右上/左下。找教堂打黑色兜袍怪爆鑰匙，去教堂選擇回血或回藍
Next -> 單獨禁閉室
}

[Solitary Confinement / 單獨禁閉室]
{
4-11 一條路，可用狼撲穿過柵欄。打死王等黑衣幽魂對話回船
Next -> 阿拉塔斯
}

[Arastas / 阿拉塔斯]
{

# ────────── Interlude (間章) ──────────


[Ashen Forest / 灰燼森林]
{
間章林間空地-1 往左側走，下一關入口在左側
}

[Scorched Farmlands / 火噬農地]
{
間章庇護所-1 先往左下開記錄點，然後右下打 BOSS，繼續右下進下一關
}

[The Khari Crossing / 卡里交匯道]
{
間章卡里市集-1 出門右直走幹掉 2 個王，回城對話蘇芮拿 +2 天賦書。最上方有熔岩神殿可拿任務物品加最大生命。沿邊往左進賈萊關口開傳送。市集營地左出一直左進卡塔爾之塘，下走左轉進下關 提示: +2 天賦書/5% 最大生命
}

[Kriar Village / 庫萊亞村]
{
